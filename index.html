What can she do?

KawaiiBot is a kawaii bot that makes your server more kawaii! Packed with multiple reaction commands, funny image commands and so much more. even porn, but we don't talk about that. The bot takes usage of Slash commands so all of the commands the bot has to offer is already provided by Discord's UI whenever you add the bot.

Enabling NSFW

If you really want to enable NSFW, you'll have to manually allow it on your server. KawaiiBot is not able to know if a channel is marked as NSFW or not, so to make it work, the bot pushes the NSFW commands without access to anyone in the server, except for admins. Follow the video to see how you can enable it on your server. The commands are called /danbooru, /gelbooru, /rule34 and /e621.
